# Receipt Scanner

Scan receipts, extract data with AI, track expenses.

## Tech Stack

- **Next.js 16** with App Router
- **React 19** + TypeScript
- **Tailwind CSS 4**
- **Shadcn** (radix-maia style)
- **Hugeicons**
- **Bun** runtime

## Quick Start

```bash
cd projects/receipt-scanner
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)

## Available Components

Shadcn UI components installed:

- AlertDialog, Badge, Button, Card
- Combobox, DropdownMenu, Field
- Input, InputGroup, Label
- Select, Separator, Textarea

Add more with:

```bash
bunx shadcn@latest add <component>
```
